# ESP32-Based Smart Kitchen Automation and Monitoring System

## Overview

This project was developed as my undergraduate capstone project in Electrical and Electronics Engineering. It presents the design and implementation of an intelligent smart kitchen system that leverages embedded systems and IoT technologies to improve safety, environmental monitoring, and automation within kitchen environments.

The system utilizes the ESP32 microcontroller alongside multiple sensors to monitor temperature, humidity, gas leakage, and human presence while providing real-time alerts and system status updates.

## Features

* Real-time temperature and humidity monitoring.
* Gas leakage detection and alert system.
* Human motion detection using a PIR sensor.
* OLED-based display of system information.
* Intelligent alert mechanism using a buzzer.
* Embedded control and automation capabilities.
* Power management using a DC-DC boost converter.
